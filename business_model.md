# 대구 지원사업 메일링 서비스 모델

## 한 줄 제안

대구 사업자에게 매일 오전 9시, 업종과 회사 상황에 맞는 정부지원사업을 선별해 보내고 지원서 초안까지 제공하는 B2B 구독 서비스.

## 초기 상품

- 지역: 대구 우선
- 기본형 가격: 월 30,000원 자동결제
- 기본형 제공물:
  - 매일 09:00 전후 맞춤 공고 메일
  - 원하는 카테고리/업종/키워드 회원 수정
  - 신규 공고 상단 강조
  - 지원금, 총사업비, 선정규모, 기업당 추정 지원금 정리
  - 마감 14일 이내 우선 알림
  - 원문 링크, 첨부파일, 다음 액션
- 프리미엄 가격: 월 100,000원
- 프리미엄 제공물:
  - 유망 공고별 지원서 초안
  - 월간 지원사업 검토 리포트
  - 전자세금계산서 발행
  - 필요 시 신청서 검수/서류 체크리스트 제공

## 고객 세그먼트

- 제조공장: 스마트공장, 로봇, 자동화, R&D, 시제품, 공정개선
- 도소매업: 소상공인, 판로, 온라인 전환, 마케팅, 라이브커머스
- 꽃/플라워: 소상공인, 로컬브랜드, 디자인, 전시, 체험, 관광 연계
- 로봇: R&D, 실증, ABB, AI, 제조 자동화, 스타트업 패키지
- 아트/콘텐츠: 미디어아트, 미디어파사드, 영상, 전시, 콘텐츠 제작지원

## 무료 홈페이지 전략

- `site/index.html`을 매일 자동 갱신해 공개 리포트로 사용
- 공개 페이지는 광고 영역을 두고, 구독자는 광고 없는 UI 제공
- 공고 클릭 시 바로 외부 사이트로 이동하지 않고 상세 팝업을 먼저 제공
  - 지원금/총사업비/선정규모/대상/마감 정보를 우리 화면에서 확인
  - 마음에 드는 공고만 원문 새창 열기
  - 원문 주소 복사 버튼 제공
- 공개 페이지는 일부 공고와 요약을 보여주고, 상세 맞춤 추천과 초안은 유료 메일에서 제공
- 업종별 키워드 페이지를 늘려 검색 유입 확보
  - 대구 제조업 지원사업
  - 대구 로봇 지원사업
  - 대구 콘텐츠 제작지원
  - 대구 소상공인 판로 지원사업

## 운영 포인트

- 무료 공개 리포트: 트래픽과 신뢰 확보
- 유료 메일: 시간 절약, 놓침 방지, 지원서 초안 가치 제공
- 첫 판매 대상은 이미 지원사업을 찾고 있는 대구 소상공인/창업기업/콘텐츠기업
- 초기에는 자동 분류 + 사람이 최종 검수하는 하이브리드가 안전함
